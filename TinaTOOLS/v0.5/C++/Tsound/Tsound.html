Documentation Librarie Tina_TOOLS (C++) - Tsound.hpp

  • Home

Disponibilité - Librairie nécessaire

  • Disponibilité : Windows
  • Librairie : Librarie Standar Du Language C++
  • Compilation : g++ -o test.exe test.cpp -lwinmm

Description de ce fichier :

  • Description : permet de regroupper des fonction pour jouer du song en arrière plan

Méthode, Function (Directive préprocesseur)

  • ErrorPlayingAudioFile
  • Tsound_Play_Back_infinit()



ErrorPlayingAudioFile

Note :

Cette directive de préprocesseur permet de retourner la valeur 34 si on arrive pas a lire le song



Tsound_Play_Back_infinit()

Note :

Cette Méthode permet de jouer des song a l'infini en arrière plan cette méthode jou que des musique en .wav

Exemple d'utilisation :

#include "Tina_TOOLS/Tsound.hpp"

int main(){
    Tsound sk;
    sk.Tsound_Play_Back_infinit("test.wav");// que des musique en .wav   
    return 0; 
}
