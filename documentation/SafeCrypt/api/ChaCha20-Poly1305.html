ChaCha20-Poly1305

SafeCrypt implements the ChaCha20-Poly1305 algorithm, a modern symmetric encryption protocol that provides security, speed, and resistance to side-channel attacks.

  • key generation
  • encryption
  • decryption
  • Key size support: 256 bits
