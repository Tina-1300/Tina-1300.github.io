AddKeyRegister()

Paramètre : cle_registre - Attend le chemin du registre .

Paramètre : key - Attend la cle a cree dans le registre.

Paramètre : value - Attend la valeur a passer a la cle.

Note :

Cette Methode permet de cree des cle de registre dans (HKEY_CURRENT_USER) de type REG_SZ

Attention :

Utiliser les registre de windows avec soing

Exemple d'utilisation :

import Thelio_REG

cle_registre = r"Software\Microsoft\Windows\CurrentVersion\Run"
key = "MonProgramme"
value = r"C:\Chemin\Vers\MonProgramme.exe"

g = Thelio_REG()
g.AddKeyRegister(cle_registre, key, value)

AddKeyRegister_ALL()

Paramètre : cle_registre - Attend le chemin du registre.

Paramètre : key - Attend la cle a ajouter dans le registre.

Paramètre : value - Attend la valeur a ajouter a la cle dans le registre.

Paramètre : number - Attend un nombre pour déterminer dans quelle section placer la cle de registre si ses dans HKEY_USERS ou autre.

Note :

Cette Methode permet de d'ajouter des cle de registre dans (HKEY_USERS, HKEY_CURRENT_USER, HKEY_LOCAL_MACHIN) de type REG_SZ par défaut le paramètre number = 0x0 si il et laisser a 0x0 il return False après il y a le 0x48843214 qui lui permet d'ajouter dans HKEY_CURRENT_USER ensuit il y a 0x48844111 qui lui permet d'ajouter dans HKEY_LOCAL_MACHIN et enfin il y a 0x48844112 qui lui permet d'ajouter dans HKEY_USERS

Attention :

Utiliser les registre de windows avec soing

Exemple d'utilisation :

import Thelio_REG

cle_registre = r"Software\Microsoft\Windows\CurrentVersion\Run"
key = "MonProgramme"
value = r"C:\MonProgramme.exe"

# nombre pour ajouter ou suprimer via les registre 
hkey_current_user = 0x48843214
hkey_local_machin = 0x48844111
hkey_users = 0x48844112

g = Thelio_REG()
if g.AddKeyRegister_ALL(cle_registre, key, value, hkey_current_user) != 400:
    print("Aucune Error Et Survenue")
else:
    print("Une error et survenue")
