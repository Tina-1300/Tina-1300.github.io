Documentation Librarie Tina_TOOLS (php) - Tina_Mysql.php

  • Home

Disponibilité - Librairie nécessaire

  • Disponibilité : Windows, linux
  • Librairie : Librarie Standar Du Language php

Description de ce fichier :

  • Description : permet de regroupper des fonction pour géré les BDD... (à la version 0.4 gérera la récupération de donner dans une bdd)

Méthode, Function, etc...

  • tin_requestSQL()



tin_requestSQL()

Paramètre : $servername_Tina - Attend l'Adresse du serveur MySQL.

Paramètre : $username_Tina - Attend le nom d'utilisateur MySQL.

Paramètre : $password_Tina - Attend le Mot de passe MySQL.

Paramètre : $database_Tina - Attend le nom de la bdd.

Paramètre : $$table_sql - Attend le nom de la table sql spécifier dans la bdd.

Paramètre : $columns - Attend les colones de la table à spécifier

Paramètre : $interrogation - Attend le nombre de valeur a transmettre pour la requette préparer.

Paramètre : $param_send - Attend les paramètre a envoyer, fait y dans l'ordre ou vous voulez qu'il soi dans les colone pour ne pas. vous perdre

Paramètre : $url_red - Attend l'url ou le fichier .php ou autre sur le quelle rediriger une fois que la requet a bien été traiter.

Note :

Cette Methode permet d'envoyer des donner a une bdd MySQL sans injection sql possible

Attention :

Si vous repere des bug ou des injection SQL possible, veuiller prévenir le createur de la lib pour patcher la vulnérabilité au plus vite

Exemple d'utilisation :

<?php
require "Tina_TOOLS/Tina_Mysql.php";

$servername_TINA = "localhost"; // Adresse du serveur MySQL
$username_TINA = "root"; // Nom d'utilisateur MySQL
$password_TINA = ""; // Mot de passe MySQL
$database_TINA = "github"; // Nom de la bdd

    if (isset($_POST['login'], $_POST['password'])){
        $login = $_POST['login'];
        $password = $_POST['password'];

        Tina_Mysql::tin_requestSQL(
            $servername_TINA, // Adresse du serveur MySQL
            $username_TINA, // Nom d'utilisateur MySQL
            $password_TINA, // Mot de passe MySQL
            $database_TINA, // Nom de la bdd
            'user', // Table de la bdd spécifier
            'login, password', // colone spécifier 
            '?, ?', // nombre de paramètre spécifier pour la requête a preparer 
            "$login, $password", // paramètre a envoyer
            "https://github.com/login" // url sur la quelle rediriger 
        );
    }else{
        echo TCF_R_ERROR; // return 211 qui veut dire (Tous les champs du formulaire doivent être remplis)
    }
?>
