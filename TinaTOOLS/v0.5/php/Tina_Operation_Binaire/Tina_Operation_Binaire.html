Tina_Operation_Bits()

Paramètre : string $nb1 - Attend une chaîne de caractères qui et le nombre binaire a calculer.

Paramètre : string $nb2 - Attend une chaîne de caractère qui et le nombre binaire a calculer avec le premier paramètre.

Paramètre : string $type_operation_enter - Attend une chaîne de caractère qui et le type d'opération a effectuer

Note :

Cette Methode permet de faire des calcule binaire telle que xor, and, or

Exemple d'utilisation :

<?php
require "Tina_TOOLS/Tina_Operation_Binaire.php";


$Tina_opp = new Tina_Operation_Binaire();

// Opération XOR
echo $Tina_opp->Tina_Operation_Bits("0110", "1010", "xor"); 

// Opération AND
echo $Tina_opp->Tina_Operation_Bits("0110", "1010", "and");

// Opération OR
echo $Tina_opp->Tina_Operation_Bits("0110", "1010", "or"); 

?>
