Les Pronoms Interrogatifs

Pronom Interrogatif Interroge sur
Who ? = Qui Une / des personnes
Where ? = Où Le lieu
Where ... from ? = d'où Le pays / la ville d'origine
When ? = Quand le moment
Whose + nom ? = à qui Le possesseur
Why ? = pourquoi la cause
What ? = que, qu'elle, quoi, quesque Une chose
Which ? = le quelle, la quelle, les quelle Une sélection entre 2 choses
What colour ? = de quelles couleurs La couleur
What ... for ? = dans quelle but Le but
What do / does ... do ? = quelle métier Le métier
What ... like ? = comment physiquement l'aspect (chose / personne)
How + be ? = comment vas-tu L'état de santé
How old / tall / long / wide ? L'âge, taille, longeur, largeur
How often ? = à quelle fréquence La fréquence
How many + nom pluriel ? = combien Le nombre
How much ? = combien Le prix ou la quantité (indénombrable)
How well ? = Comment La compétence
