Read_File_Result_To_Binary()

Paramètre : string $NameFile - Attend une chaîne de caractères qui et le nom du fichier a lire.

Note :

Cette Methode permet de lire le contenu d'un fichier en mode binaire

Exemple d'utilisation :

<?php
require "Tila_TOOLS/Tina_Binary_Read_File.php";

$Tina_Read_Bin = new Tina_Binary_Read_File();
echo $Tina_Read_Bin->Read_File_Result_To_Binary("test.txt"); 

?>
