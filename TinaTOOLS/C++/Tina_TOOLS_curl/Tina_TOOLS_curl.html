Documentation Librarie Tina_TOOLS (C++) - Tina_TOOLS_curl.hpp

  • Home

Disponibilité - Librairie nécessaire

  • Disponibilité : Windows
  • Librairie : Librarie Standar Du Language C++ & librairie curl
  • Compilation : g++ -o test.exe test.cpp -lcurl

Description de ce fichier :

  • Description : permet de regroupper des fonction pour utiliser la librairie CURL plus facilement

Méthode, Function (Directive préprocesseur)

  • ERROR_EMPTY_URL
  • UE
  • Tina_Get_STATUS_HTTP_URL()
  • Tina_Download_File()



ERROR_EMPTY_URL

Note :

Cette Directive de préprocesseur permet de retourner 32 si l'url et vide



UE

Note :

Cette Directive de préprocesseur permet de retourner -1 si il ne connais pas l'erreur



Tina_Get_STATUS_HTTP_URL()

Paramètre : std::string URL - Attend une chaîne de caractères qui est l'URL.

Note :

Cette Methode permet de récupérer le statu http d'une url

Exemple d'utilisation :

#include <iostream>
#include "Tina_TOOLS/Tina_TOOLS_curl.hpp"

int main() {
    Tina_TOOLS_curl tincurl;
    std::cout << tincurl.Tina_Get_STATUS_HTTP_URL("https://trial-and-error-blu.000webhostapp.com/p.wav") << std::endl;
    return 0; 
}



Tina_Download_File()

Paramètre : const std::string& url - Attend une chaîne de caractères qui est l'URL.

Paramètre : const std::string& outputFile - Attend une chaîne de caractères qui et le fichier a télécharger

Note :

Cette Methode permet de télécharger un fichier, musique, filme etc...

Exemple d'utilisation :

#include <iostream>
#include "Tina_TOOLS/Tina_TOOLS_curl.hpp"

int main() {
    Tina_TOOLS_curl tincurl;
    tincurl.Tina_Download_File("https://trial-and-error-blu.000webhostapp.com/p.wav", "p.wav");
    return 0; 
}
