addButtonkvasheilds()

Paramètre : $text - Attend une chaîne de caractères qui et le test pour le button.

Paramètre : $type = "button" - Attend une chaîne de caractère qui et le type de button, par défau a button.

Note :

Cette Methode permet de preparer le button et avec du text sur le button et un mode que l'on peut entre entre button, submit et reset

Exemple d'utilisation :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <?php 
        require "Tina_TOOLS/ButtonGenerator.php";
        $buttonGen = new ButtonGenerator();
        $buttonGen->addButtonkvasheilds("Envoyer", "submit");
        echo $buttonGen->renderButtonskvasheilds(); // cette function permet d'afficher le button 
    ?>
</body>
</html>
