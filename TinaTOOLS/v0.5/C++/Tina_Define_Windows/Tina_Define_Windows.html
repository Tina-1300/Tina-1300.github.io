Tina_Console_UTF8

Note :

Cette Directive de préprocesseur permet d'afficher en console un text en UTF8

Exemple d'utilisation :

#include <iostream>
#include <windows.h>
#include "Tina_TOOLS/Tina_Define_Windows.hpp"

int main(){
    Tina_Console_UTF8;
    std::cout << "Je suis là je suis présent a l'hôpitale" << std::endl;
    return 0; 
}
